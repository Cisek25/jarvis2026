## 🔴 AKCJE KLIENTA — w panelu IdoBooking (URGENT)

### 1. SEO — odblokować indeksację (P1, P0 critical)
**Problem**: Cała witryna ma `<meta name="robots" content="noindex, nofollow">`. Google nie zindeksuje żadnej podstrony.

**Kroki**:
1. Panel IdoBooking → **Konfiguracja → SEO**
2. Znaleźć opcję "Strona w wersji deweloperskiej" / "Niewidoczna dla wyszukiwarek"
3. **Odznaczyć**.
4. Verify: `curl -s https://client58360.idobooking.com/ | grep robots` powinno zwrócić `index, follow` lub brak meta robots.

### 2. Title DE homepage (P3)
**Problem**: `<title>` na DE = "Wypełnij to pole tytułem twojej strony wizytówki" (placeholder z CMS).

**Kroki**:
1. Panel → **Konfiguracja → Strona wizytówki → Język DE → Title**
2. Wpisać: `Apartments zur Miete in Breslau | Fair Rentals`
3. Meta description: `Familienunternehmen. 21 Apartments in Breslau — Altstadt, Zentrum, Kępa Mieszczańska. Booking 9.6, Google 4.7. Direkt buchen.`

### 3. Title /contact (P4)
**Problem**: `<title></title>` całkowicie pusty.

**Kroki**:
1. Panel → **Treści → Strony tekstowe → Kontakt (PL)**
2. Wypełnić Title: `Kontakt | Fair Rentals — Wrocław`
3. Powtórzyć dla EN: `Contact | Fair Rentals — Wrocław`
4. DE: `Kontakt | Fair Rentals — Breslau`

### 4. Menu EN — dodać ATTRACTIONS (P5)
**Problem**: Menu EN: GO TO TOP / CONTACT / OFFER / CORPORATE HOUSING / SHORT-TERM RENTAL MANAGEMENT. Brakuje "ATTRACTIONS".

**Kroki**:
1. Panel → **Wygląd → Menu → Język EN**
2. Dodać pozycję: "ATTRACTIONS" → link do `/en/txt/200/Attractions` (lub odpowiedni angielski URL)
3. Pozycja MIĘDZY "OFFER" a "CORPORATE HOUSING"

### 5. Menu DE — "CONTACT"→"KONTAKT" + dodać "VERMIETUNGSVERWALTUNG" (P6)
**Problem**: Menu DE używa angielskiego "CONTACT" (powinno być "KONTAKT"). Brakuje pozycji "Obsługa najmu" (po niemiecku "VERMIETUNGSVERWALTUNG" lub "MIETVERWALTUNG").

**Kroki**:
1. Panel → **Wygląd → Menu → Język DE**
2. Edytuj pozycję "CONTACT" → "KONTAKT"
3. Dodać pozycję: "VERMIETUNGSVERWALTUNG" → link do `/de/txt/203/Vermietungsverwaltung`

### 6. Meta description per oferta (P18)
**Problem**: Każda oferta ma meta description = description homepage'a.

**Kroki**:
1. Panel → **Oferty → kolejna oferta → SEO Meta Description**
2. Napisać unikalny opis (2-3 zdania, ~150 znaków) dla każdej oferty.
3. Przykład dla Golden Hour: `Apartament Golden Hour w centrum Wrocławia — sauna, siłownia, parking. Booking 9.6. Idealny dla par i biznesu.`

### 7. OG image https + aktualizacja "21 apartamentów" (P19, P20)
**Problem**: OG image używa `http://` (mixed content). OG description mówi "19" zamiast "21".

**Kroki**:
1. Panel → **Konfiguracja → SEO → Open Graph**
2. OG image: zmienić `http://...` → `https://...`
3. OG description: zaktualizować z "19" na "21" apartamentów.

### 8. Tel: w footerze — usunąć spację (P13)
**Problem**: System generuje `<a href="tel:+48 575092755">` (ze spacją). Standard E.164 wymaga BEZ spacji.

**Kroki**:
1. Panel → **Konfiguracja → Dane kontaktowe**
2. Sprawdź pole "Telefon" — usuń ewentualne spacje (powinno być: `+48575092755`)
3. Wyświetlanie z spacjami robi się przez CSS/HTML, nie w danych źródłowych.
