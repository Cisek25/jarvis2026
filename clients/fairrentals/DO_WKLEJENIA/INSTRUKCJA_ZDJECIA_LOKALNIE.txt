Mozna tymczasowo:
1. Pobrac zdjecia z Unsplash w wysokiej rozdzielczosci

   (Unsplash License pozwala na to: free to use, no attribution required)
3. Podmienic linki w HTML

To eliminuje zewnetrzna zaleznosc, ale nie zmienia zdjec na unikalne.

P27 — LOGO: ZALECENIA OPTYMALIZACJI
================================================================

Aktualny logo: https://client58360.idobooking.com/images/owner/wideLogo.svg

Sprawdzenia do wykonania:
1. CZY plik to SVG? (preferowane) Czy PNG/JPG?
   - SVG = idealny: skalowalny, lekki (<5 KB), ostry na retinie
   - PNG = zalecane jesli SVG niemozliwy
   - JPG = NIE polecane dla logo (artefakty, brak alpha)

2. Rozmiar pliku:
   - SVG: <5 KB
   - PNG: <20 KB
   - Jesli wiekszy — kompresja w https://tinypng.com lub https://svgomg.net

3. Wymiary loga:
   - SVG: zachowac aspect ratio, viewBox 0 0 X Y
   - PNG: 2x rozdzielczosc dla retina (np. 240x80 displayed, 480x160 file)

4. Test:
   Otworz https://client58360.idobooking.com/ w Chrome
   F12 → Network → przeladuj
   Sprawdz Size kolumne dla loga — powinno byc <20 KB

JAK WGRAC NOWE LOGO:
Panel → Wyglad → Logo / Tozsamosc wizualna
→ Wgraj plik SVG (lub PNG)
→ Zaznacz "Wyswietlaj na naglowku"
→ Zapisz
