── KROK 1: CSS ────────────────────────────────────────────
Panel → Wygląd → Edytor CSS
→ USUŃ cały dotychczasowy CSS
→ Wklej zawartość: CSS_EDYTOR.css
→ Zapisz


── KROK 2: STRONA GŁÓWNA ─────────────────────────────────
Panel → Wygląd → Strona główna → Edytor treści (Tryb HTML)
→ Wklej zawartość: GLOWNA_PL__body_top.html
→ Zapisz


── KROK 3: GLOBAL JS (body_bottom) ────────────────────────
Panel → Wygląd → Nagłówek i Stopka → Koniec sekcji BODY
→ Wklej zawartość: KONIEC_BODY.html
→ Zapisz

⚠️  WAŻNE: Wklej do body_bottom, NIE do body_top!
    CMS stripuje <script> z body_top.


── KROK 4: PODSTRONY (/txt/) ──────────────────────────────
Panel → Treść → Podstrony → Utwórz nową podstronę

  /txt/201/ → "O nas"        → ONAS_PL__body_top.html
  /txt/202/ → "Galeria"      → GALERIA_PL__body_top.html
  /txt/203/ → "Lokalizacja"  → LOKALIZACJA_PL__body_top.html
  /txt/204/ → "Atrakcje"     → ATRAKCJE_PL__body_top.html
  /txt/205/ → "Regulamin"    → REGULAMIN_PL__body_top.html
  ⚠️  /kontakt/ → NIE MA pliku HTML — stylujemy CSS-only!
       Klient nie ma dostępu do body_top kontaktu.

Dla każdej podstrony:
→ Tytuł: jak powyżej
→ Edytor treści → Tryb HTML → Wklej odpowiedni plik
→ Zapisz


── KROK 5: MENU NAWIGACJI ─────────────────────────────────
Panel → Wygląd → Menu

Kolejność linków w menu:
  1. Apartamenty     → /offers
  2. O nas           → /txt/201/
  3. Galeria         → /txt/202/
  4. Lokalizacja     → /txt/203/
  5. Atrakcje        → /txt/204/
  6. Kontakt         → /txt/206/

⚠️  Regulamin (/txt/205/) → dodaj w stopce, nie w głównym menu.
⚠️  Przycisk "REZERWUJ" jest dodawany automatycznie przez JS.


── KROK 6: WYRÓŻNIONE OFERTY ─────────────────────────────
Panel → Oferty → Wyróżnij oferty z lokalizacji Piekary 1-3
→ Zaznacz oferty, które mają się pojawić na stronie głównej
→ JS automatycznie zbuduje karty z .container-hotspot


── KROK 7: SLIDER / ZDJĘCIA ──────────────────────────────
Panel → Wygląd → Slider
→ Dodaj 2-3 zdjęcia kamienicy / Starego Miasta
→ Rekomendacja: 1920x1080 lub większe, horyzontalne
→ Tymczasowo: zdjęcia Torunia z Wikimedia (600px)


── KROK 8: REDIRECT HOMEPAGE ──────────────────────────────
⚠️  KRYTYCZNE: Usunąć redirect na osadahelenow.pl z CMS!
Panel → Wygląd → Strona główna → Sprawdź redirect


── KROK 9: DOMENA ─────────────────────────────────────────
Panel → Ustawienia → Domeny
→ Dodaj apart-torun.pl jako domenę własną
→ Ustaw DNS: CNAME na engine23326.idobooking.com
→ Po propagacji: cała strona działa pod apart-torun.pl

⚠️  Po mapowaniu domeny — booking modal (iframe) automatycznie
    używa silnika pod tą samą domeną. Brak przekierowań!


── KROK 10: LOGO ──────────────────────────────────────────
Panel → Wygląd → Logo
→ Wgraj logo od projektanta (PNG/SVG, transparentne tło)
→ Logo musi dobrze wyglądać w dwóch wariantach:
  - Białe (na ciemnym hero — filter:invert CSS)
  - Kolorowe (na jasnym scrolled header)
